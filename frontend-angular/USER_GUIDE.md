# VK Parser User Guide

## 📋 Overview

The VK Parser is a comprehensive web application for managing VK groups, monitoring keywords, analyzing comments, and controlling parsing operations. This guide will help you navigate and use all features effectively.

## 🚀 Getting Started

### First Time Setup

1. **Access the Application**

   - Open your web browser
   - Navigate to the application URL
   - You'll be redirected to the login page

2. **Login**

   - Enter your username and password
   - Click "Login" to access the dashboard
   - If you don't have an account, contact your administrator

3. **Dashboard Overview**
   - The dashboard provides an overview of all system activities
   - View key metrics and recent activities
   - Quick access to all main features

## 🎯 Main Features

### 📊 Dashboard

The dashboard is your central hub for monitoring system activity and performance.

**Key Elements:**

- **System Status**: Real-time status of parsing operations
- **Recent Activity**: Latest actions and events
- **Performance Metrics**: System performance indicators
- **Quick Actions**: Fast access to common tasks

**How to Use:**

1. View system status at the top of the dashboard
2. Check recent activities in the activity feed
3. Monitor performance metrics in real-time
4. Use quick action buttons for common tasks

### 👥 Groups Management

Manage your VK groups for monitoring and analysis.

**Features:**

- View all connected VK groups
- Add new groups for monitoring
- Edit group settings and preferences
- Remove groups from monitoring
- Search and filter groups

**How to Use:**

#### Viewing Groups

1. Navigate to "Groups" in the main menu
2. View all groups in the table
3. Use search to find specific groups
4. Use filters to narrow down results

#### Adding a New Group

1. Click "Add Group" button
2. Enter the VK group URL or ID
3. Configure group settings:
   - **Name**: Display name for the group
   - **Category**: Group category for organization
   - **Monitoring Level**: How frequently to check
   - **Keywords**: Specific keywords to monitor
4. Click "Save" to add the group

#### Editing Groups

1. Find the group in the table
2. Click the "Edit" button (pencil icon)
3. Modify the settings as needed
4. Click "Save" to update

#### Removing Groups

1. Find the group in the table
2. Click the "Delete" button (trash icon)
3. Confirm the deletion

### 🔍 Keywords Management

Organize and manage keywords for monitoring across groups.

**Features:**

- Create and organize keywords
- Categorize keywords for better management
- Bulk import/export keywords
- Monitor keyword performance
- Search and filter keywords

**How to Use:**

#### Adding Keywords

1. Navigate to "Keywords" in the main menu
2. Click "Add Keyword" button
3. Enter keyword details:
   - **Keyword**: The word or phrase to monitor
   - **Category**: Organize keywords by category
   - **Priority**: Set monitoring priority
   - **Description**: Optional description
4. Click "Save" to add the keyword

#### Bulk Import

1. Click "Import Keywords" button
2. Upload a CSV file with keywords
3. Map columns to keyword fields
4. Review and confirm import

#### Managing Categories

1. Use the category filter to view keywords by category
2. Create new categories as needed
3. Organize keywords into logical categories

### 💬 Comments Management

Monitor and manage comments from VK groups.

**Features:**

- View all monitored comments
- Advanced search and filtering
- Bulk moderation actions
- Comment analytics
- Export comment data

**How to Use:**

#### Viewing Comments

1. Navigate to "Comments" in the main menu
2. View comments in the table
3. Use search to find specific comments
4. Apply filters to narrow results

#### Advanced Search

1. Use the search form to find specific comments:
   - **Text**: Search in comment text
   - **Author**: Search by comment author
   - **Group**: Filter by VK group
   - **Date Range**: Filter by date
   - **Keywords**: Filter by keywords
2. Click "Search" to apply filters

#### Bulk Actions

1. Select multiple comments using checkboxes
2. Choose an action from the bulk actions menu:
   - **Mark as Reviewed**: Mark comments as reviewed
   - **Export**: Export selected comments
   - **Delete**: Remove comments from monitoring
3. Confirm the action

### ⚙️ Parser Controls

Control parsing operations and monitor system status.

**Features:**

- Start/stop parsing operations
- Monitor parsing status in real-time
- Configure parsing settings
- View parsing statistics
- Manage parsing schedules

**How to Use:**

#### Starting Parsing

1. Navigate to "Parser" in the main menu
2. Check current parsing status
3. Click "Start Parsing" to begin operations
4. Monitor progress in real-time

#### Stopping Parsing

1. Click "Stop Parsing" to halt operations
2. Confirm the action
3. Wait for graceful shutdown

#### Configuration

1. Click "Settings" to configure parser
2. Adjust settings as needed:
   - **Interval**: How often to check for new content
   - **Groups**: Which groups to monitor
   - **Keywords**: Which keywords to track
   - **Filters**: Content filtering rules
3. Click "Save" to apply changes

Configure application settings and preferences.

## ⌨️ Keyboard Shortcuts

### Navigation

- `Ctrl + 1`: Dashboard
- `Ctrl + 2`: Groups

- `Ctrl + 5`: Parser
- `Ctrl + 6`: Monitoring
- `Ctrl + 7`: Settings

### Actions

- `Ctrl + S`: Save
- `Ctrl + F`: Search
- `Ctrl + E`: Export
- `Ctrl + I`: Import
- `Esc`: Close dialog/cancel

### Table Operations

- `Space`: Select row
- `Enter`: Edit row
- `Delete`: Delete selected
- `Ctrl + A`: Select all
